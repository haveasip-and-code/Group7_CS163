Yaanbao
One of the [Arts od Boabom]. ; In this course, the student learns movements using various implements, such as short staffs, long ones, etc. ; Through this Art the student (who has already advanced somewhat in Osseous Boabom) studies the perfection of movement, solidity, and security in both technique and themselves, developing a new set of skills while using these implements as an extension of their own body. ; Successively, these elements change, both in shape and in length.
