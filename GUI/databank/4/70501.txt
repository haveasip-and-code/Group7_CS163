ABM
Technical [jargon]. ; 1. A [TLA] for 'Anything But Microsoft'. ; 2. A doctrine of using any operating system, development or application software to acheive a goal as long as the software is not designed by Microsoft nor, for some people, a Microsoft branded release of an otherwise adequate software title. ; 3. The notion that Microsoft products are a stability risk, security risk and/or overpriced to be worth paying for or using. ; ABM's, ABMer
