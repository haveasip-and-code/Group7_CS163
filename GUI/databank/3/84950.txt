abroach
To set abroach; to let out, as liquor; to broach; to tap. [Obs.] Chaucer.\n\n1. Broached; in a condition for letting out or yielding liquor, as a cask which is tapped. Hogsheads of ale were set abroach. Sir W. Scott. 2. Hence: In a state to be diffused or propagated; afoot; astir. \"Mischiefs that I set abroach.\" Shak.
